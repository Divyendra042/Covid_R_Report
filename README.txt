COVID-19 Data Analysis in R
----------------------------

This project performs an exploratory data analysis (EDA) on the global COVID-19 dataset from Our World In Data.

1. Download the dataset from:
   https://covid.ourworldindata.org/data/owid-covid-data.csv

2. Place it in the same directory as the covid_analysis.R file.

3. Run the R script to generate the following plots:
   - Total COVID-19 Cases Over Time (India, US, Brazil)
   - Top 10 Countries by Total Deaths (Bar Chart)
   - Vaccination Progress Over Time

4.Libraries used:
- tidyverse
- ggplot2
- lubridate

5.Visualizations
Time Series Plot – Total COVID-19 cases for India, United States, and Brazil
Bar Plot – Top 10 countries by total deaths
Line Plot – Vaccination progress over time by country
